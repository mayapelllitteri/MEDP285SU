How To Make A Crochet Shrimp!
Follow below for step by step instructions:

FIRST:

chain 7

ROW 1:

Sk 1st ch from hook. 6 sl st in 2 of each ch.

ROW 2

Ch 1, (2 sc)

ROW 3:

Ch 1, turn, (2 sc), sc

ROW 4:

Ch 1, turn, 3 sc evenly

ROW 5:

Ch 1, turn, (2 sc), 2 sc

ROW 6-8:

Ch 1, turn, 4 sc evenly

ROW 9:

Ch 1, turn, sl st, sc, 2 hdc

ROW 10:

Ch 2, turn, hdc, sc, sk1 sl st

ROW 11:

Ch 1, turn, sl st, sc, hdc

ROW 12:

Ch 2, turn, hdc, sc, sl st

ROW 13:

Ch 1, turn, sl st, sc, hdc

ROW 14:

Ch 2, turn, hdc, sc, sl st

ROW 15:

Ch 1, turn, sl st, sc, hdc

ROW 16:

Ch 1, turn, 3 sc evenly

Cut yarn, leaving at least a yard of thread

REPEAT 1-16 to make second half

Whip stich halves together

Add stuffing inside

TAIL:

Ch 2, htr, ch2, sl st, repeat

Add eyes of choice

DONE!

MATERIALS:
  1. Orange / coral yarn
  2. G/4.0mm crochet hook
  3. Scissors
  4. Tapestry needle
  5. Small buttons & thread OR black half rounds & super glue (for eyes)
  6. cotton stuffing

ABBRIVIATIONS:
  • "ch" = chain
  • "sk" = skip
  • "sl st" = slip stitch
  • "( )" = in the same loop
  • "sc" = single crochet
  • "hdc" = half double crochet
  • "htr" = half treble crochet
